#Movie App

A modern React-based movie application built for learning and portfolio purposes. This project includes authentication forms, theme switching, reusable components, and a clean and scalable structure.

---

##  Features

-  Login & Register forms with validation  
-  Dark / Light theme support  
-  Reusable components (Input, Layout, etc.)  
-  Custom hooks for form handling  
-  Input validation (email, password rules)  
-  Responsive design  
-  Clean and modular React structure  

---

##  Tech Stack

- React.js  
- JavaScript (ES6+)  
- Context API  
- Custom Hooks  
- Tailwind CSS / CSS  
- Git & GitHub  

---

##  Project Structure

src/
components/   Reusable components
contexts/     Theme context
hooks/        Custom hooks
pages/        Login, Register, etc.
layouts/      Layout components
utils/        Validation helpers

---

##  How to Run

1. Clone the repository

git clone https://github.com/KEREMABBASZADE/movie_app.git

2. Install dependencies

npm install

3. Run the project

npm run dev
